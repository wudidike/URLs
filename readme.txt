## 使用简介：

点击exe即可使用，上传需要检测txt文件即可检测，检测完成后可选择保存路径和重命名，默认命名检测结果.xlsx

### **主要功能**：

1. **批量检测 URL 的 HTTP 状态码**：
   - 支持 HTTP 和 HTTPS 协议。
   - 能够检测常见的 HTTP 状态码，如 200（成功）、404（未找到）、500（服务器错误）等。
   - 对于无法访问的 URL，输出对应的错误信息。
2. **获取页面标题**：
   - 使用 `requests` 库获取页面响应，解析 HTML 页面获取 `<title>` 标签内容。
   - 如果页面没有标题，输出 "null"。
3. **智能处理 URL**：
   - 自动检测并补充缺失的协议头（如 `http://` 或 `https://`）。
   - 如果 URL 中包含端口号，脚本会支持端口检测。
4. **支持通过代理服务器进行检测**：
   - 如果需要，脚本能够使用代理进行 URL 的访问，确保在特定网络环境下的检测。
5. **错误处理和超时控制**：
   - 自动捕捉网络请求中的异常，如超时、连接错误等，并记录在输出中。
   - 如果 URL 无法访问，会显示相关错误信息，而不是卡住脚本。
6. **Excel 输出结果**：
   - 检测的结果会被保存到一个 Excel 文件中，列出 URL、状态码、标题以及可能的错误信息。
   - 方便后续分析、存档或者分享给他人。
